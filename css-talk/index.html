Best Practices

  • Never use ID selectors. Classes all the way.
  • Avoid attaching classes to element type selectors.
  • Avoid !important.
  • Always use vendor prefixes.
    Put plain property last.
  • Provide for a fallback experience.
  • Use CSS Lint
